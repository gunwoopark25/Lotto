# 로또 번호 생성기 (Lotto Number Generator)

역대 로또 당첨번호 데이터를 기반으로 통계적 가중치(statistical weight)를 적용하여 번호를 예측·생성하는 Python 도구입니다.

---

## 주요 기능

- 과거 당첨번호 CSV/엑셀 파일을 불러와 빈도(frequency) 분석
- **역빈도(inverse-frequency)** 가중치 세트 3개: 적게 출현한 번호에 높은 확률 부여
- **빈도(frequency)** 가중치 세트 2개: 많이 출현한 번호에 높은 확률 부여
- 최근 5회 당첨번호에 5% 불이익(penalty) 적용
- 사용자가 구간별(1~10, 11~20, 21~30, 31~40, 41~45) 번호 개수를 직접 지정 가능
- 총 5세트의 번호를 생성하여 출력

## 사용 방법

### 방법 1: 명령행 인자(command-line argument)로 파일 경로 전달

```bash
python lotto_generator.py "역대 로또 번호.csv"
```

### 방법 2: 실행 후 파일 경로 직접 입력

```bash
python lotto_generator.py
# 실행 후 프롬프트에 파일 경로 입력
```

### 실행 흐름

1. CSV/엑셀 파일 로드 → 회차별 당첨번호 파싱
2. 전체 빈도 계산 및 최근 5회 번호 추출
3. 구간별 번호 개수 입력 (총합 반드시 6)
4. 가중치 기반 번호 생성 → 5세트 출력

## 알고리즘 상세

### 가중치 계산 (Weight Calculation)

각 번호에 대해 정규화(normalization) 후 선형 변환을 적용합니다. 결과 범위는 [0.7, 1.3]입니다.

**역빈도 가중치 (inverse-frequency weight):**

```
w_n = 0.7 + (1 - (f_n - f_min) / (f_max - f_min)) * 0.6
```

**빈도 가중치 (frequency weight):**

```
w_n = 0.7 + ((f_n - f_min) / (f_max - f_min)) * 0.6
```

여기서 `f_n`은 번호 `n`의 전체 출현 횟수입니다. 최근 5회 출현 번호에는 추가로 `× 0.95` 패널티(penalty)가 적용됩니다.

### 구간 내 비복원 추출 (Sampling without Replacement)

각 구간 내에서 `numpy.random.choice`로 가중치 기반 비복원 추출(weighted sampling without replacement)을 수행합니다.

## 입력 데이터 형식

CSV 파일은 아래와 같은 2줄 헤더 구조를 따릅니다:

| 행   | 내용                                                       |
|------|------------------------------------------------------------|
| 1행  | 회차 \| 당첨 숫자 (병합셀)                                 |
| 2행  | (공백) \| 1 \| 2 \| 3 \| 4 \| 5 \| 6 \| 보너스             |
| 3행~ | 회차번호 \| 번호1 ~ 번호6 \| 보너스번호                    |

- 보너스(bonus) 번호는 분석에서 제외됩니다.
- 지원 형식: `.csv`, `.xlsx`, `.xls`
- 최신 회차가 상단에 있어도 내부적으로 자동으로 오래된 순서로 정렬합니다.

## 주의사항

이 도구는 통계적 경향성에 기반한 **번호 제안 도구**입니다. 로또는 독립 시행(independent trial)이므로 과거 데이터가 미래 당첨번호를 보장하지 않습니다.
